# Django проект для автоматизации сдачи-приёмки строительных работ
## ООО «Мостоотряд-69»

Система предназначена для автоматизации процесса создания, учёта и экспорта актов КС-2 и справок КС-3.

---

## 📋 Описание функционала

### Основные возможности:
- **Управление контрагентами** - создание и редактирование информации о подрядчиках и заказчиках
- **Управление объектами строительства** - учёт строительных объектов
- **Управление договорами** - ведение реестра договоров подряда
- **Создание актов КС-2** - формирование актов о приёмке выполненных работ
- **Генерация справок КС-3** - создание справок о стоимости работ
- **Экспорт в XML** - выгрузка документов в формате XML по Приказу ФНС № ЕД-7-26/691
- **Валидация XML** - проверка файлов на соответствие XSD схеме
- **Статусы актов** - отслеживание статуса (черновик, на согласовании, подписан, оплачен)

### Статусы актов:
- 📝 **Черновик** (draft) - акт создан, но ещё не отправлен
- 🔄 **На согласовании** (review) - акт отправлен на согласование
- ✅ **Подписан** (approved) - акт подписан всеми сторонами
- 💰 **Оплачен** (paid) - работы по акту оплачены
- ❌ **Отклонён** (rejected) - акт отклонён

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
cd mo69_diplom
pip install -r requirements.txt
```

### 2. Применение миграций

```bash
python manage.py migrate
```

### 3. Создание суперпользователя

```bash
python manage.py createsuperuser
```

Введите:
- Имя пользователя
- Email (необязательно)
- Пароль

### 4. Запуск сервера разработки

```bash
python manage.py runserver
```

Сервер запустится по адресу: http://127.0.0.1:8000/

## 🔐 Доступ к системе

### Вход в систему
1. Откройте http://127.0.0.1:8000/login/
2. Введите имя пользователя и пароль
3. Нажмите "Войти"

### Админ-панель
1. Откройте http://127.0.0.1:8000/admin/
2. Войдите под учётной записью суперпользователя
3. Управляйте данными через удобный интерфейс

## 📊 Работа с системой

### Создание акта КС-2

1. Перейдите в раздел **"Создать акт"**
2. Заполните основную информацию:
   - Номер акта
   - Дата акта
   - Договор
   - Объект
   - Период выполнения работ
   - Ставка НДС
3. Добавьте строки выполненных работ:
   - Нажмите **"Д Добавить строку"**
   - Укажите наименование, количество и цену
4. Сохраните акт

### Просмотр акта

1. Откройте дашборд
2. Кликните на акт в таблице или нажмите кнопку 👁️
3. Просмотрите информацию об акте
4. Скачайте документы в форматах PDF или XML

### Экспорт документов

Из карточки акта доступны кнопки скачивания:
- 📄 **КС-2 (PDF)** - акт о приёмке выполненных работ
- 📄 **КС-3 (PDF)** - справка о стоимости работ
- 📝 **КС-2 (XML)** - XML файл для систем электронного документооборота
- 📝 **КС-3 (XML)** - XML файл справки КС-3

### Валидация XML

1. Перейдите в раздел **"Валидация XML"**
2. Загрузите XML файл
3. Нажмите **"Проверить валидность"**
4. Получите результат проверки

## 🛠 Технические детали

### База данных
- **SQLite** - используется по умолчанию для простоты развёртывания
- Файл БД: `db.sqlite3`

### Технологии
- **Django 5.2** - веб-фреймворк
- **Bootstrap 5.3** - фронтенд фреймворк
- **lxml** - работа с XML
- **Pillow** - обработка изображений

### Кодировка XML
- XML файлы генерируются в кодировке **windows-1251**
- Соответствует требованиям Приказа ФНС № ЕД-7-26/691
